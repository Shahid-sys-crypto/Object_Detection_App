# 🎯 Object Detection App

A simple Object Detection application built using **YOLOv5**, **PyTorch**, and **OpenCV**.  
This app allows users to detect objects either from an image file or through a live webcam feed.

---

## 📌 Features

- Detect objects from an image  
- Real-time object detection using webcam  
- Uses pre-trained YOLOv5 model  
- Fast and accurate detection  

---

## 🛠️ Technologies Used

- Python  
- PyTorch (`torch`)  
- OpenCV (`cv2`)  
- NumPy
